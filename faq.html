📱 手機與網頁問題
網頁打不開怎麼辦？+
確認手機有連網路（WiFi 或行動數據）。網址請重新輸入： https://empiresun36.github.io/activity/ 不要用舊的書籤。
網頁打開是空白或顯示錯誤？+
清除瀏覽器快取後重新開啟。
  1. iPhone：設定 → Safari → 清除歷史記錄與網站資料
  2. Android：Chrome → 設定 → 清除瀏覽資料
若還是舊版本，在網址後加 ?v=5 強制更新
沒有智慧型手機怎麼辦？+
由工作人員協助，用工作人員的手機代為掃描集章，最後截圖憑證給住戶保存。
📷 掃描問題
按掃描按鈕出現「Permission denied」？+
需要開啟相機權限：
  1. iPhone：設定 → Safari → 相機 → 選「允許」
  2. Android：設定 → 應用程式 → Chrome → 權限 → 相機 → 允許
設定完後回到網頁重新整理再試。
相機開了但一直掃不到？+
  1. 確認 QR Code 立牌是白底黑色（不是黑底白色）
  2. 光線要充足，避免反光
  3. 距離保持 15～25 公分
  4. 手機保持穩定，等待 2～3 秒
改用手機內建相機直接掃 QR Code，掃到後點網址連結，一樣可以集章
掃到 QR Code 但沒跳出集章成功？+
確認掃的是本次活動的立牌。如果確定是活動立牌，請重新整理網頁再試一次。
🗺️ 集章問題
集章紀錄不見了？+
集章記錄存在手機瀏覽器，以下情況會導致遺失：
  • 更換瀏覽器
  • 清除瀏覽器資料
  • 換手機
請住戶用同一台手機同一個瀏覽器完成活動。若真的無法恢復，由工作人員現場酌情處理。
同一個 QR Code 重複掃了？+
網頁會顯示「已經集過這個囉！」，不會重複計算，不影響集章進度，不用擔心。
某個地點的 QR Code 立牌不見或損壞？+
馬上通知現場工作人員，由工作人員用自己的手機幫住戶補登該地點。
集章數不滿10個可以兌獎嗎？+
正常情況下需集滿10個。
☔ 若當日遇大雨取消戶外地點，不滿10個亦可兌獎，以現場公告為準
🎁 兌獎問題
忘記截圖憑證怎麼辦？+
回到活動網頁，若集章紀錄還在，可重新點「領取完成憑證」再截圖。若資料已遺失，由工作人員現場酌情處理。
如何防止重複兌獎？+
建議工作人員採用以下方式：
  1. 打開 Google 試算表報名資料
  2. 找到該住戶的戶號
  3. 兌獎後在旁邊欄位標記「✅已領取」
每戶以戶號為準，同一戶號只能兌換一次
如何判斷憑證是否被偽造？+
核對憑證上的防偽碼格式（SUN-日期-電話尾碼-亂數），以及戶號與電話是否與 Google 試算表報名資料相符。
一戶多個小朋友，可以換多份嗎？+
每戶每人限換乙份，以戶號為準。
禮品發完了怎麼辦？+
向住戶致歉說明，登記住戶資料，活動結束後另行安排補發或補償。
☔ 雨天問題
下雨了，活動還辦嗎？+
輕雨照常進行。大雨時取消戶外掃描點，兌換地點改至大廳，集章數不滿10個亦可兌獎。
怎麼通知大家改變地點？+
  1. 透過社區 LINE 群組發通知
  2. 在原兌換地點（花園）張貼告示
  3. 引導住戶至大廳
📊 資料收集問題
如何查看所有報名者資料？+
  1. 打開 Google 表單
  2. 點「回覆」頁籤
  3. 點右上角綠色試算表圖示
即可看到所有姓名、戶號、電話，即時更新。
活動結束後資料怎麼處理？+
建議活動結束後將 Google 試算表下載備份，並於適當時間刪除表單，保護住戶個資。
🚨 緊急狀況
網站突然掛掉怎麼辦？+
GitHub Pages 偶有維護，通常幾分鐘內恢復。
備用方案：工作人員手動在紙上記錄各地點打卡，集滿蓋章換禮
某地點 QR Code 無法掃描？+
工作人員至該地點確認立牌狀況，若損壞則移除，並公告該地點取消，其餘地點集章數到達即可兌獎。
有住戶投訴不公平怎麼處理？+
耐心說明活動規則，若有特殊情況由主辦委員現場裁量，以和諧為原則。
